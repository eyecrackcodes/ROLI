## Overview

All agents operate under one unified model. There are no T1/T2/T3 tier distinctions for operational purposes. Every agent receives the same inbound lead allocation, has access to the leads pool, and is held to the same pipeline and conversion standards.

---

## Daily Lead Allocation

| Item | Target |
|---|---|
| Inbound leads per day | 7 |
| Lead cost | $60 per lead |
| Daily conversion target | 1 deal/day (≈14% CR) |
| Additional leads | Available if CR ≥ 15% and pipeline is clean |

Agents who are compliant with pipeline discipline and whose close rate supports it may request additional leads beyond 7/day.

---

## Leads Pool

The leads pool is available to all agents. Pool activity is **supplemental** to inbound production, not a replacement for it.

| Metric | Target |
|---|---|
| Pool follow-ups per day | Find 5 people to follow up with |
| Pool sales per week | 1 sale from pool contacts |
| Self-assign rate | ≥ 30% of pool answered calls |

**Pool workflow:**
1. Enter pool session during designated blocks (after follow-up appointments are complete).
2. Dial pool leads — these are shared, unassigned contacts.
3. Self-assign every answered contact (even not-interested / DNC) to remove from rotation.
4. Set follow-up appointments for interested contacts.
5. Target: find 5 people per day worth following up with.

---

## Pipeline Discipline

Pipeline management is the highest-priority operational discipline. A clean pipeline protects lead investment and prevents revenue rot.

| Metric | Standard |
|---|---|
| Max pipeline size | 30 leads |
| Past due follow-ups | 0 (always) |
| Follow-up appointments | Work all scheduled appointments before any pool or inbound activity |

**Priority order when pipeline is unhealthy:**
1. Work past-due follow-ups immediately
2. Clear stale pipeline leads (close out or withdraw)
3. Resume normal inbound / pool activity

## Daily Schedule (CST)

| Block | Time | Activity |
|---|---|---|
| 1 | 9:00 – 9:30 | Work follow-up appointments (scheduled callbacks) |
| 2 | 9:30 – 12:00 | Inbound calls + pool sessions |
| 3 | 12:00 – 1:00 | Lunch |
| 4 | 1:00 – 4:30 | Inbound calls + pool sessions |
| 5 | 4:30 – 5:00 | Pipeline cleanup, notes, next-day prep |

Follow-up appointments take priority over all other activity. If you have scheduled callbacks, work them at the scheduled time before entering the pool or taking inbound calls.

---

## Close Rate Thresholds

| Threshold | Value | Action |
|---|---|---|
| Target | ≥ 15% | On track — eligible for additional leads |
| Floor | 10% | Warning — focus on converting existing pipeline |
| Crisis | < 7% for 3+ days | Quality review — audit call quality and presentations |

Close rate is calculated weekly as total sales / total leads delivered.

---

## Action Recommendations

The system monitors each agent hourly and generates recommendations:

| Priority | Condition | Recommendation |
|---|---|---|
| P1 | Past due > 0 | **WORK FOLLOW-UPS** (critical) |
| P2 | Pipeline > 30 | **CLEAR PIPELINE** (critical) |
| P3 | CR < 7% for 3+ days | **REVIEW QUALITY** (critical) |
| P4 | CR < 10% | **WORK FOLLOW-UPS** (warning) |
| P5 | Pool assigns < 5 at midday | **GET IN POOL** (warning) |
| P6 | CR ≥ 15%, pipeline clean | **TAKE MORE LEADS** (info) |
| P7 | All clear | **ON TRACK** (info) |

|---|---|
| Guaranteed minimum | $52,000/year |

The guaranteed minimum applies if commissions do not reach the monthly equivalent of $52k/year. Commission is calculated as 30% of placed premium.

## Missed Calls / Leads Pool Follow-Ups

All missed inbound calls are sent to agents. Agents should:
1. Review missed call list daily
2. Find 5 contacts per day from pool / missed calls worth following up with
3. Set follow-up appointments for qualified contacts
4. Target 1 additional sale per week from pool/missed call follow-ups
